   SPEECH-EMOTION-PROBLEM

*  This repository contains the speech emotion classification problem for the MIDAS internship task.
*  It contains the predicted values of the dataset.
*  Weights have also been uploaded in this repository for testing any external audio files.
*  No data files have been committed to this repository.
*  This code was trained on Google Colab.
*  The bash script to create an environment and download required dependencies has been uploaded(createEnv).
   ***** To run the script, open terminal, go to the location of the file and execute command "bash createEnv" *****

*  In case task number two is inaccessible, it has been uploaded here (bluekernel). It contains no data file
